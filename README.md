## Usage

After cloning it

### Go into the project directory

```bash
cd Links
```

### Create `.env` file in the root of the project and provide the variables

#### Expected env file

```bash
DB_URI=
```

### Run this command

```bash
pnpm i && pnpm build && pnpm start
```

### Open `http://localhost:3000` and enjoy
